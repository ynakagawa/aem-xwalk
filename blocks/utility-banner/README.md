# Header Utility Banner (Courtesy Banner)

The **Header Utility Banner**, also known as the **Courtesy Banner**, is an informational header component designed to appear above the Mega Navigation on a website. Its primary purpose is to improve the user experience by offering immediate access to essential utilities and providing timely updates or critical messages.

## Features

- Displays **above the Mega Navigation**
- Offers quick access to:
  - **Sign In**
  - **Find a Store**
  - **My Account**
- Supports site-wide:
  - **Announcements**
  - **Promotional Messages**
  - **Critical Information**

## Purpose

To enhance user engagement and accessibility by ensuring key utilities and important updates are always visible and readily available at the top of the page.

## Usage

Include the Header Utility at the top of your webpage layout, directly above the Mega Navigation component. Ensure the content is relevant, concise, and updated dynamically based on the site's announcements and user utilities.

## Variation

1) Presently the **Utility Banner** supports **space-around** variation, allowing space around different sections in the banner maintained as a flexbox.
